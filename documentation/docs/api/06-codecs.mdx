# API

## ReactiveCodec

```dart title="Declaration"
abstract interface class ReactiveCodec {
  String mimeType()
  Uint8List encode(dynamic input)
  dynamic decode(Uint8List input)
}
```

## ReactiveMessagePackCodec

```dart title="Declaration"
class ReactiveMessagePackCodec implements ReactiveCodec {
  dynamic decode(Uint8List input)
  Uint8List encode(dynamic input)
  String mimeType() 
}
```
